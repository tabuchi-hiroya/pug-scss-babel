# webpack・babel環境

## ディレクトリ構成

* **dist**：製品版
  * css
  * scripts
  * index.html
* **src**：開発用
  * css
  * scripts
  * index.html
* **.babelrc**：babel設定ファイル
* **.gitignore**：git除外設定ファイル
* **bs-config.js**：BrowserSync設定ファイル
* **package.json**：パッケージ管理ファイル
* **webpack.config.js**：webpack設定ファイル
